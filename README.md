# Global BMI Data Analysis (R)

## Project Overview

This project analyzes global BMI (Body Mass Index) trends using R. The
goal is to explore obesity and underweight patterns across countries,
clean and validate the dataset, and produce meaningful visual insights.

------------------------------------------------------------------------

## Objectives

- Analyze global obesity and underweight rates
- Perform data cleaning and validation
- Identify top countries by obesity prevalence
- Generate visualizations for exploratory data analysis
- Prepare insights for interpretation and reporting

------------------------------------------------------------------------

## Technologies & Tools Used

- R
- tidyverse
- ggplot2
- plotly
- dplyr
- tidyr

## How to Run the Project

1. Install required R packages: install.packages(c("tidyverse",
   "ggplot2", "plotly"))

2. Place the dataset inside the `/data` folder.

3. Open `analysis.R` and ensure the dataset path is:
   data/bmi_dataset.csv

4. Run the script in RStudio.
